# One-page portfolio (GitHub Pages)

## Files
- `index.html` — single page portfolio
- `styles.css` — minimal styling

## GitHub Pages (quick)
1. Create or use a repo (recommended user-site repo name: `octaviusleo.github.io`)
2. Upload these files to the repo root
3. In GitHub: Settings → Pages → deploy from `main` branch (root)

## Custom domain (later)
After you buy the domain, add it in GitHub Pages Settings → Custom domain.
Then create DNS records at your registrar (A records for apex + CNAME for `www`).
